

# yelp-camp
A simple Web development project using Node.js, express, EJS and mongoDB.

## Features
* Use of **Bootstrap**
* Admin Roles
* Pagination
* Fuzzy search
* Google Maps
* Passport & session-based authorization
* RESTful Routing
